## 选择运算符

选择运算符(@C)或(@choose)查找二项式系数。 该运算符等效于Binomial(n,r)函数。

## 举例  
[在Mathstudio上浏览](http://mathstud.io/?input[0]=ayBAY2hvb3NlIDI%3D&input[1]=ayBAYyAy)



>   ```math
>   k @choose 2
>   ```
>   $\frac { k ( k \cdot 1 ) } { 2 }  $

>   ```math
>   k @c 2
>   ```
>   $\frac { k ( k \cdot 1 ) } { 2 }  $
