> 1 + 2 + 3 + 4 + 5
> ```
>
> $15$

> ```math
> 15!
> ```
>

### 代数运算

> ```math
> x + x + x
> ```
>
> $3x$

> ```math
> Expand((a + b)^10)
> ```
>
> $a^{10}+b^{10}+10 a^{9} b+10 a b^{9}+45 a^{8} b^{2}+45 a^{2} b^{8}+120 a^{7} b^{3}+120 a^{3} b^{7}+210 a^{6} b^{4}+210 a^{4} b^{6}+252 a^{5} b^{5}$

> ```math
> Factor(x^4 + 5x^2 - 6)
> ```
>
> $(x-1)(x+1)(x^2+6)$

> ```math
> Apart((x^2) / (x^2 + 1)^2)
> ```
>
> $-\frac{1}{(x^2+1)^2}+\frac{1}{x^2+1}$

### 单位换算

> ```math
> 3@miles -> @kilometers
> ```
>
> **4.83km**

> ```math
> @cups -> @tablespoons
> ```
>
> **16tbsp**

> ```math
> @megabyets -> @bytes
> ```
>
> **1 048 576byets**

> ```math
> 10@liters -> @gallons
> ```
>
> **2.64gallons**
