### 特殊值
特殊值(例如$\pi$)，可以使用 "@" 符号作为前缀进行表示。

### 圆周率
> ```math
> @pi
> ```
> $\pi$
>
> *3.14159265359*

> ```math
> +@pi
> ```
> $3.14159265359$

> ```math
> +(2@pi)
> ```
> $6.28318530718$

### 自然常数
> ```math
> @e
> ```
> $e$
> *2.718281828459*

> ```math
> +@e
> ```
> $2.718281828459$

### 欧拉常数
> ```math
> @gamma
> ```
> $\gamma$
> *0.577215664902*

> ```math
> +@gamma
> ```
> $0.577215664902$

### 圆常数（圆周率的二倍）
> ```math
> @tau
> ```
>
> $\tau$
>
> *6.28318530718*

> ```math
> +@tau
> ```
>
> $6.28318530718$

### 虚数单位
> ```math
> @i
> ```
>
> $i$

> ```math
> sqrt(-1)
> ```
>
> $i\sqrt{1}$

> ```math
> abs(3 + 2@i)
> ```
>
> $3.605551275464$

### 科学计数法
> ```math
> 3@E5
> ```
>
> $300000$

> ```math
> 12@E-6
> ```
>
> $0.000012$

### 角度
> ```math
> sin(30@d)
> ```
>
> $\frac{1}{2}$
>
> *0.5*

> ```math
> cos(45@d)
> ```
>
> $\frac{1}{\sqrt{2}}$
>
> *0.707106781187*

> ```math
> 360@d
> ```
>
> $2\pi$
>
> *6.28318530718*

### 无穷
> ```math
> @inf
> ```
>
> $\infty$

### 复无穷
> ```math
> @ComplexInfinity
> ```
>
> **ComplexInfinity**

### 未定元（一个大小不能确定的数值量）
> ```math
> @Indeterminate
> ```
>
> **Indeterminate**
